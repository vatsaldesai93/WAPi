# WAPi
### Wireless Access Point (Hotspot) for Raspberry Pi

### Notes
- Tested to work on Raspberry Pi 3 (Raspbian OS Stretch Lite)
- May also work on Debian-like distributions with minor edits
- Uses Hostapd and Dnsmasq

### Requirements
  * Python-Netfilter
  * Python-Pathlib
  * Python-Netifaces
  * Python-Shutil
